# Pacman-Concept-3JS

A 3D Pacman concept game built using Three.js, featuring volumetric lighting effects, bloom post-processing, and classic Pacman gameplay in a 3D environment.

## Features

- 3D maze navigation with Pacman character
- Ghost AI with chasing behavior
- Collectible dots and power pellets
- Volumetric light rays for atmospheric effects
- Bloom effects for enhanced visuals
- Smooth animations using GSAP TweenMax
- WebGL shader-based rendering

## How to Run

1. Clone the repository:
   ```
   git clone https://github.com/BinaryVortex/Pacman-Concept-3JS.git
   ```

2. Open `index.html` in a modern web browser that supports WebGL.

## Technologies Used

- **Three.js**: 3D graphics library
- **WebGL Shaders**: Custom vertex and fragment shaders for lighting and effects
- **EffectComposer & UnrealBloomPass**: Post-processing effects
- **Perlin Noise**: Procedural generation
- **MeshLine**: 3D line rendering
- **GSAP TweenMax**: Animation library
- **HTML5 Canvas**: Rendering context
